<h3 class="text" style="font-size:30px;"> Pseudo-Classes and Links </h3> <br>

<ul>
    <li> Recall that classes are user-defined selectors that can be used to control individual HTML element formatting. </li>
    <li> Pseudo-classes are defined by a colon and are not user-defined. Specific pseudo-classes can be used on specific HTML elements only. </li>
    <li> The anchor element has four pseudo-classes: <b> <i> link, visited, hover and active. </i> </b> Each one affects the behavior of the anchor element. </li>
    <li> The <b> <i> link </i> </b> pseudo-class is for <b> <i> hyperlinks </i> </b> that have <b> <i> NOT </i> </b> yet visited. </li>
    <li> The <b> <i> visited </i> </b> pseudo-class is for <b> <i> visited hyperlinks. </i> </b> </li>
    <li> The <b> <i> hover </i> </b> pseudo-class is for <b> <i> hyperlinks </i> </b> when the mouse is <b> <i> hovered </i> </b> over them. </li>
    <li> The <b> <i> active </i> </b> pseudo-class is for <b> <i> hyperlinks </i> </b> when they are <b> <i> clicked. </i> </b> </li>
</ul>

<h4> Pseudo-Classes and Links Property & Syntax </h4> 
<ul>
    <li> The order in which the pseudo-classes are stated is important. </li> 
    <li> Take note of the <b> LVHA </b> or <b><u>L</u></b>ink, <b><u>V</u></b>isited, <b><u>H</u></b>over, <b><u>A</u></b>ctive is the order of the pseudo-classes. </li>
</ul>

<p> <b> Note: </b> The value that you place inside {…} will be displayed whenever a pseudo-class property is performed. You can set the color of the text or element, its background, its decoration and many other characters. </p>

<table border="1">
   <tr>
      <th> Property </th>
      <th> Value </th>
      <th> Description </th>
   </tr>
   <tr>
      <td> :link </td>
      <td> a:link {…} </td>
      <td> Sets unvisited link characteristic. </td>
   </tr>
   <tr>
      <td> :visited </td>
      <td>  a:visited {…} </td>
      <td> Sets visited link characteristic or effect. </td>
   </tr>
   <tr>
      <td> :active </td>
      <td> a:active {…} </td>
      <td> Sets the character of an active link element as when the user clicks the mouse on the link. </td>
   </tr>
   <tr>
      <td> :hover </td>
      <td> a:hover {…} </td>
      <td> Sets the character of a link element as whenthe user hovers or moves the mouse over the link. </td>
   </tr>
   <tr>
      <td> :focus </td>
      <td> a:focus {…} </td>
      <td> Sets the character of an active link element as when the user clicks and moves the mouse on the link. </td>
   </tr>
</table> <br>

<ul>
    <li> Be reminded that the hover property should be placed after the link property for it to function properly. </li>
    <li>  Pseudo-classes can also be used on any element or block elements on the web page, especially the hover property, so that, whenever one hovers over an image, paragraph or any element, you can set properties to make it more interactive. Example is changing         the font color, setting a background, border and many other properties. </li>
